This folder has 6 sample images. 
The SLIDES subfolder has the files converted to BMPs which can be loaded on a MicroSD card
and shown with LackPaint.
They wre created by running
  make_slides SLIDES
from a command prompt.

Refer to the comments in make_slides.bat

SizingActions.png shows the different ways the batch can resize JPGs, portrait or landscape
to fit the LCD screen
